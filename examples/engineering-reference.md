# Engineering Reference (example)

`update_agents_epic_table.py` regenerates a per-epic progress table in place,
between the two marker comments below. Put this file at
`docs/guides/contributing/engineering-reference.md` in your target repo (or edit
`AGENTS_PATH` in the script to wherever you keep it), keep the markers, and run:

```bash
python scripts/ticket_hygiene/update_agents_epic_table.py
```

Everything between the markers is overwritten; text outside them is left alone.
