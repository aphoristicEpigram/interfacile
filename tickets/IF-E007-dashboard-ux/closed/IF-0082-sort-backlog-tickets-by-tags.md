# IF-0082 · Sort backlog tickets by tags

## Context

Backlog tickets sorted by date, risk, priority or effort - but not by the
tags the cards now carry.

## Approach

A 'by tags' option in the tickets sort select, ordering by the same weight
the grid's TAGS column uses (blocked 8 / decision 4 / wip 2 / pinned 1),
heaviest first.

## Acceptance criteria

- [x] The backlog tickets select offers 'by tags' and orders blocked and decision work to the top.
