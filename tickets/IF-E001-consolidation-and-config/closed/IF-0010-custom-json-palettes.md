# IF-0010 · Custom JSON palettes

Shipped: `_palette_css` generates a full `:root` override from a small set of
semantic roles (light + dark), a repo can set `"theme"` to any of 14 built-in
presets or a fully custom palette, and interfacile itself runs a custom green
"acid" theme. Backgrounds are the per-interface context cue.
